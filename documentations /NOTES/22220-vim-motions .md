# vim-motions

## Moving the cursor

- h : moves to the left
- l : moves to the right
- k : moves up
- j : moves down

## Exiting vim

- :q! : exits the vim , DISCARDING any changes made .
- :qa : exits the vim .
- :wqa : exits the vim , after saving the changes made.
